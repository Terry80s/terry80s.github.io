[Mac]ignore mac os update

2021-01-17

打开”系统偏好设置”,点”软件更新”.

取消选择”自动保持我的Mac最新”

点击”高级”按钮

取消所有的勾选、点”好”

退出系统设置,打开终端命令行,分别执行如下3步操作:

sudo softwareupdate --ignore "macOS Big Sur"

defaults write com.apple.systempreferences AttentionPrefBundleIDs 0

killall Dock

恢复

sudo /usr/sbin/softwareupdate --reset-ignored
